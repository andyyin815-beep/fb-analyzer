找出在 Facebook 中最關心你的朋友

這個工具可以透過分析 Facebook 的頁面原始碼，找出最常與你互動或最關注你的朋友清單。

※ 請使用在自己的臉書帳號，不要擅自或強迫查看他人的，以免造成隱私、法律或感情問題。※

操作步驟

  1. 請稍候...
  2. 打開你的 Facebook
    → 鼠標右鍵點擊頁面的空白處 → 選擇「查看網頁原始碼」
    打開你的 Facebook
    → 按 ⌘ + U
    打開你的 Facebook
    → 按 Option + ⌘ + U → 單擊代碼區
    打開你的 Facebook
    → 按 Ctrl + U
    打開你的 Facebook
    → 按 Ctrl + U → 單擊代碼區
  3. 在原始碼頁面全選 (Ctrl + A) → 複製 (Ctrl + C)

🎉 分析結果

此頁面與 Facebook 無關。Facebook 是 Facebook 公司的註冊商標。
